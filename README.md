# gobof.js

6dof tracking inside the browser!

![](small.gif)

open firefox,
go to:
about:config
then set
network.websocket.allowInsecureFromHTTPS
flag to true.

Then go to:

https://jaroslaw-weber.github.io/gobofjs/

# how to use

- set fps (default to 60)
- input websocket server and click update button
- click start tracking
